oneHTML docu officiel

oneHTML est un programme qui permet de crée une documentation dans UN fichier html depuis un UN fichier markdown.

Utilisation

Il vous suffira de télécharger la dernière version du programme dans le repository Github officiel. Puis de lancer le code HTML

Pour plus d'information sur le fonctionnement du programme, rendez-vous sur le README
